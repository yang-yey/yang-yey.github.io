12、表单

  • input：

    • type 输入类型：text password submit radio(单选) checkbox(多选) button

      ​ H5新：color；date；datetime；datetime-local；email；month；

      ​ number；range；search；tel；time；url；week

    • value ： 初始值

    • readonly ： 输入字段只读（不可修改） readonly 或者 readonly=”readonly”

    • disabled： 输入字段禁用 （不可用、不可点击、不会被提交）
